<h1 id="migrating-from-1-x-to-2-x">Migrating from 1.x to 2.x</h1> <h2 id="new-features">New features</h2> <ul> <li>You can now pass to <code>router.add()</code> objects containing <code>canActivate</code> functions and those functions will be registered. No need to call for each route <code>addNode</code> or <code>canActivate</code>.</li> <li>Persistent parameters plugin now available.</li> </ul> <h2 id="breaking-change">Breaking change</h2> <ul> <li><code>router.registerComponent</code> and <code>router.deregisterComponent</code> have been removed in favour of <code>canDeactivate</code></li> <li>Additional arguments now apply to middleware functions.</li> <li>Context has been removed from middleware functions.</li> <li>Middleware functions are now a function taking a router instance and returning a function called on each transition.</li> <li>Plugins, like middleware functions, are now a function taking a router instance and returning an object of methods (which doesn&#39;t contain an <code>init</code> function anymore).</li> <li><code>autoCleanUp</code> is no longer shared with <em>router5-listeners</em>. If you need to turn off automatic deregistration of node listeners, pass <code>{ autoCleanUp: false }</code> to the listeners plugin.</li> <li><code>router5</code> package now exports <code>Router5</code> as default, and <code>RouteNode</code>, <code>loggerPlugin</code>, <code>errCodes</code> and <code>transitionPath</code> as named exports</li> </ul> <h2 id="code-example">Code example</h2> <p><strong>ES2015+</strong></p> <pre><code class="lang-javascript javascript">import Router5, { loggerPlugin } from &#39;router5&#39;; import historyPlugin from &#39;router5-history&#39;; import listenersPlugin from &#39;router5-listeners&#39;; const router = new Router5() .add([{ name: &#39;home&#39;, path: &#39;/home&#39; }]) .usePlugin(historyPlugin()) .usePlugin(listenersPlugin()) // Development helper .usePlugin(loggerPlugin()) .start(); </code></pre> <p><strong>ES5</strong></p> <pre><code class="lang-javascript javascript">var router5 = require(&#39;router5&#39;); var Router5 = router5.default; var loggerPlugin = router5.loggerPlugin; var historyPlugin = require(&#39;router5-history&#39;); var listenersPlugin = require(&#39;router5-listeners&#39;); var router = new Router5() .add([{ name: &#39;home&#39;, path: &#39;/home&#39; }]) .usePlugin(historyPlugin()) .usePlugin(listenersPlugin()) // Development helper .usePlugin(loggerPlugin()) .start(); </code></pre>
