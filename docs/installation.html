<h1 id="getting-started">Getting started</h1> <blockquote> <p><em>router5</em> is available in all major formats: <strong>ES6</strong>, <strong>CommonJS</strong>, and <strong>UMD</strong>.</p> </blockquote> <p>It can be installed using <strong>npm</strong> or <strong>yarn</strong>. Alternatively, you can download a specific version from <a href="https://github.com/router5/router5/releases">github</a>.</p> <h2 id="installation">Installation</h2> <pre><code class="lang-sh"># yarn yarn add router5 # npm npm install router5 </code></pre> <h2 id="include-_router5_-in-your-application">Include <em>router5</em> in your application</h2> <p><strong>CommonJS</strong></p> <pre><code class="lang-javascript javascript">// ES2015+ import createRouter, { RouteNode, errorCodes, transitionPath, loggerPlugin } from &#39;router5&#39;; import browserPlugin from &#39;router5/plugins/browser&#39;; import listenersPlugin from &#39;router5/plugins/listeners&#39;; import persistentParamsPlugin from &#39;router5/plugins/persistentParams&#39;; // ES5 var router5 = require(&#39;router5&#39;); var createRouter = router5.default; var RouteNode = router5.RouteNode; var errorCodes = router5.errorCodes; var constants = router5.constants; var transitionPath = router5.transitionPath; var loggerPlugin = router5.loggerPlugin; var browserPlugin = require(&#39;router5/plugins/browser&#39;); var listenersPlugin = require(&#39;router5/plugins/listeners&#39;); var persistentParamsPlugin = require(&#39;router5/plugins/persistentParams&#39;); </code></pre> <p><strong>UMD</strong></p> <p>A UMD bundle is available in <code>/dist/umd</code>, and it should be used for AMD or globals. The bundle contains all <em>router5</em> dependencies (<em>route-node</em> and <em>path-parser</em>), but doesn&#39;t contain plugins.</p> <p>Plugins are packaged separately and available in <code>/dist/umd</code>:</p> <ul> <li><code>browserPlugin</code> UMD module is named <code>router5BrowserPlugin</code></li> <li><code>listenersPlugin</code> UMD module is named <code>router5ListenersPlugin</code></li> <li><code>persistentParamsPlugin</code> UMD module is named <code>router5PersistentParamsPlugin</code> bundle is named <code>router5ListenersPlugin</code>.</li> </ul>
